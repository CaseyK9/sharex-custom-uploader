# sharex-uploader
A one-file ShareX uploader. Forked from https://github.com/MrSheldon/sharex-custom-uploader.

# Setup
1. Copy `upload.php` to your webserver.
2. Change the configuration (`$key` and `$dir`).
3. Create a custom uploader in ShareX.
